# Mrror

A simple accountability tool keeping you aligned.

## Overview

Mrror is a minimal Next.js web app that helps you stay aligned with your goals through daily reflection. It asks two powerful questions:

1. **"What does a day in the life of the person you want to become look like?"** - Helps you visualize your ideal self
2. **"Did you lie to yourself today?"** - Keeps you honest with yourself

All responses are saved locally in your browser with timestamps, creating a personal journal of your journey.

## Getting Started

First, install dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the app.

## Build for Production

```bash
npm run build
npm start
```

## Tech Stack

- **Next.js 16** - React framework
- **TypeScript** - Type safety
- **Tailwind CSS v4** - Styling
- **localStorage** - Data persistence

## Project Structure

- `app/page.tsx` - Main application component with comprehensive comments
- `app/layout.tsx` - Root layout and metadata
- `app/globals.css` - Global styles and Tailwind setup
